# MDXit Project Themes

Put project-specific theme tokens here.

This directory is for user customization, not built-in runtime code. The workbench automatically loads `*.json` theme token files from here.

Recommended shape:

```json
{
  "name": "project-theme",
  "tokens": {
    "bg": "#f7f8f5",
    "panel": "#ffffff",
    "text": "#19211d",
    "accent": "#315f9a"
  }
}
```
